AngelHunt

A social aggregator that allows users to add their favorite websites and let others upvote or downvote them.

My Role

A website that I worked on for the Responsive Web Design specialization on Coursera

Tools & Methods

  • Meteor.js, HTML&CSS, Bootstrap, MongoDB
  • Responsive Web Design

    • Goal

    Create a social aggregator using Meteor.js and Responsive Web Design principle.

    Similar to how Reddit/ProductHunt works, users will need to create an account or sign in first.

    Users can add a new website to AngelHunt and describe what it does.

    All users can upvote or downvote a link on AngelHunt.

    If users click "Detail", it will take them to the detail page where they can see the comments.

    Users can submit a comment on a website besides voting.
