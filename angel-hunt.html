AngelHunt

A social aggregator that allows users to add their favorite websites and let others upvote or downvote them.

My Role

A web development project

Tools & Methods

  • Meteor.js, HTML, CSS, Bootstrap, MongoDB
  • Web Development, Responsive Web Design

    • Goal

    Create a social aggregator using Meteor.js and the Responsive Web Design principle.

    Overview

    I built this website as a side project when I was working on the Responsive Web Design Specialization on Cousera.

    Web UI

    Similar to how Reddit or ProductHunt works, when a user visits the website, he or she will need to create an account or sign in to use the website. User will see a list of websites that have been added by previous visitors, the ranking of the sites and how many upvotes or downvotes each site has.

    Users can add a new website to AngelHunt and describe what it does.

    All users can upvote or downvote a link on AngelHunt.

    If user clicks "Detail", it will take him or her to the detail page where the user can see comments on a specific site.

    Users can submit a comment on a website besides voting.
